### 产品特色

- **代码开源**：代码开源，满足用户二次自定义开发
- **高可用**：支持集群模式，提高用户冗灾能力
- **多租户**：一套系统，多部门同时使用
- **云原生**：支持docker快速部署，只需一行命令即可使用
- **私有化部署**：用户可内网部署，提高用户网络安全
- **spark原生**：纯原生spark，支持spark的所有原生用法
